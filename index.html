Containers

We moesten met CSS verschillende soorten containers maken om alles apart te kunnen bedienen qua kleuren, vorm, padding, etc.

klik om naar de container opdracht te gaan

Flexbox

We moesten met CSS een flexbox maken waarbij we logo's en daaronder tekst moesten plaatsen en als je het scherm verkleinde het mee bewoog.

klik om naar de flexbox opdracht te gaan

Navbar/tabs

Daarin moesten wij een navbar of tabs maken die naar 3 verschillende pagina's ging die we zelf moesten maken.

klik om naar de navbar/tabs opdracht te gaan

Grid

We moesten een grid maken zodat alle secties goed op de juiste plek kwamen te staan en dat je bij verschillende groten andere kleuren kan selecteren.

klik om naar de grid opdracht te gaan

Bootstrap

we moesten een kleine opdracht (in mijn geval carousel) maken waarbij we kennis maakten met Bootstrap.

klik om naar de bootstrap opdracht te gaan
